2. 函数防抖
> 适用场景：  
> 按钮提交场景：防止多次提交按钮，只执行最后提交的一次  
> 服务端验证场景：表单验证需要服务端配合，只执行一段连续的输入事件的最后一次，还有搜索联想词功能类似

```
const debounce =(fn,delay)=>{
    let timer = null
    return (...args)=>{
        if (timer)clearTimeout(timer)
        timer = setTimeout(()=>{
            fn.apply(this,args)
        },delay)
    }
}

```
3. 函数节流
> 适用场景:  
> 拖拽场景：固定时间内只执行一次，防止超高频次触发位置变动  
> 缩放场景：监控浏览器resize  
> 动画场景：避免短时间内多次触发动画引起性能问题

```
const throttle = (fn, delay) => {
    let flag = true
    return (...args) => {
        if (!flag) return
        flag = false
        setTimeout(() => {
            fn.apply(this, args)
            flag = true
        }, delay)
    }
}
```

- 利用时间戳实现

```
const throttle = (fn,delay)=>{
    let start = Date.now()
    return (...args)=>{
        let end = Date.now()
        if(end-start>=delay){
            fn.apply(this,args)
            start = Date.now()
        }
    }
}
```
